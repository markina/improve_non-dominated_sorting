    id N M
    N строчек по M double
- сбор времени работы BOS алгоритма на этих же данных (NAMETEST_N_M_time_bos.txt)
    Формат вывода:
    id N M t n
    id - соответствует id из первого запуска
    t - время в наносекундах
    n - количество запусков алгоритма
- сбор времени работы Fast алгоритма на этих данных (NAMETEST_N_M_time_fast.txt)
    Формат вывода:
    id N M t n
- агрегирование данных.
 Сравнение скоростей:
 (NAMETEST_N_M_time_result_fast_better.txt + NAMETEST_N_M_result_bos_better.txt)
    Фомат вывода:
    N M
    speed_fast (time_fast / n_fast)
    speed_bos (time_bos / n_bos)
    ----
 Статистика:
 (NAMETEST_N_M_statistic_bos.txt, NAMETEST_N_M_statistic_fast.txt)
    Формат вывода:
    N cnt
    cnt - сколько раз при количестве точек N был лучше алгоритм bos/fast
